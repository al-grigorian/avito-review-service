# Avito Review Service

Тестовое задание для стажёра Backend (осень 2025).

## Инструкция по запуску
1. Клонируйте репозиторий: `git clone https://github.com/yourusername/avito-review-service.git`
2. Запустите: `docker-compose up`

Сервис доступен на http://localhost:8080.
